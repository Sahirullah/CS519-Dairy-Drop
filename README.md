# Dairy-Drop

A MERN stack e-commerce application for dairy products.

## Features

- Product catalog with 4-column responsive grid layout
- Shopping cart functionality
- User authentication
- Admin dashboard
- Order management
- Wishlist feature
- Search functionality

## Tech Stack

- **Frontend**: React, Material-UI, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Styling**: CSS3, Responsive Design

## Getting Started

### Prerequisites
- Node.js and npm installed
- MongoDB running locally or connection string configured

### Installation

1. Clone the repository
```bash
git clone https://github.com/Sahirullah/Dairy-Drop.git
cd Dairy-Drop
```

2. Install dependencies
```bash
cd client
npm install
cd ../server
npm install
```

3. Start the development server
```bash
# From client directory
npm start

# From server directory (in another terminal)
npm start
```
